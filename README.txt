*****************************************************************************
						KLP Dashboard for Akshara
*****************************************************************************

This repository contains the following:
1) Scripts for creating the database that is used for generating the dashboard.
2) A Django application which generates displays and organises the dashboard. It includes:
	- The viewing pages which organises the dashboard view
	- The model which the application uses to generate and process the data.
	- The python code which analyses the data and stores it in the required tables
